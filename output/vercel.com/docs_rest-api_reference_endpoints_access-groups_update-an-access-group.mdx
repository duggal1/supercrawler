#### Authorizations

[​](#authorization-authorization)

Default authentication mechanism

#### Path Parameters

[​](#parameter-id-or-name)

#### Query Parameters

[​](#parameter-team-id)

The Team identifier to perform the request on behalf of.

`"team_1a2b3c4d5e6f7g8h9i0j1k2l"`

[​](#parameter-slug)

The Team slug to perform the request on behalf of.

`"my-team-url-slug"`

#### Body

[​](#body-name)

The name of the access group

`"My access group"`

[​](#body-projects)

Showchild attributes

[​](#body-projects-project-id)

The ID of the project.

`"prj_ndlgr43fadlPyCtREAqxxdyFK"`

[​](#body-projects-role)

The project role that will be added to this Access Group. "null" will remove this project level role.

`"ADMIN"`

[​](#body-members-to-add)

List of members to add to the access group.

```json
[
  "usr_1a2b3c4d5e6f7g8h9i0j",
  "usr_2b3c4d5e6f7g8h9i0j1k"
]
```

[​](#body-members-to-remove)

List of members to remove from the access group.

```json
[
  "usr_1a2b3c4d5e6f7g8h9i0j",
  "usr_2b3c4d5e6f7g8h9i0j1k"
]
```

#### Response

[​](#response-name)

The name of this access group.

`"my-access-group"`

[​](#response-created-at)

Timestamp in milliseconds when the access group was created.

`1588720733602`

[​](#response-team-id)

ID of the team that this access group belongs to.

`"team_123a6c5209bc3778245d011443644c8d27dc2c50"`

[​](#response-updated-at)

Timestamp in milliseconds when the access group was last updated.

`1588720733602`

[​](#response-access-group-id)

ID of the access group.

`"ag_123a6c5209bc3778245d011443644c8d27dc2c50"`

[​](#response-members-count)

Number of members in the access group.

`5`

[​](#response-projects-count)

Number of projects in the access group.

`2`

[​](#response-entitlements)

[​](#response-team-roles)

Roles that the team has in the access group.

```json
["DEVELOPER", "BILLING"]
```

[​](#response-team-permissions)

Permissions that the team has in the access group.

```json
["CreateProject"]
```
