## Logspot

Logspot is a lightweight, free and open source template for your changelog made with Vue, Nuxt and Tailwindcss.

How it works?

1. Each`.md`file in`/content/posts/`counts as one changelog post, logspot uses the[nuxt content module](https://content.nuxtjs.org/)

#### Features

1. Markdown support
1. Icons support using Iconify
1. Code syntax highlighting using Shiki.js
1. Frontmatter - dates, author supported
1. Vue components inside markdown using Nuxt contents MDC format - More on this[here](https://content.nuxtjs.org/guide/writing/mdc)
1. Icons support -`:icon{name="ph:user-circle-duotone"}`will show a user icon - find more icons at[https://icones.js.org](https://icones.js.org)
1. Alert component with icon, title and deion.
