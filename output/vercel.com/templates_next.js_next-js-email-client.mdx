## Next.js Email Client

This is an email client template built with Next.js and Postgres. It's built to show off some of the features of the App Router, which enable you to build products that:

* igate between routes in a column layout while maintaining scroll position (layouts support)
* Submit forms without Java enabled (progressive enhancement)
* igate between routes extremely fast (prefetching and caching)
* Retain your UI position on reload (URL state)

**Demo:[https://next-email-client.vercel.app](https://next-email-client.vercel.app)**

### Tech Stack

* **Framework**:[Next.js](https://nextjs.org/)
* **Database**:[Postgres](https://www.postgresql.org/)
* **ORM**:[Drizzle](https://orm.drizzle.team/)
* **Styling**:[Tailwind CSS](https://tailwindcss.com/)
* **UI Library**:[shadcn/ui](https://ui.shadcn.com/)
