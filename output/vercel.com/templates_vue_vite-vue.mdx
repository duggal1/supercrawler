## Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3`< setup>`SFCs, check out the[setup docs](https://v3.vuejs.org/api/sfc--setup.html#sfc--setup)to learn more.

While this project uses Vue.js, Vite supports many popular JS frameworks.[See all the supported frameworks](https://vitejs.dev/guide/#scaffolding-your-first-vite-project).

### Deploy Your Own

Deploy your own Vite project with Vercel.

[](https://github.com/vercel/examples/tree/main/)

*Live Example:[https://vite-vue-template.vercel.app](https://vite-vue-template.vercel.app)*

#### Deploying From Your Terminal

You can deploy your new Vite project with a single command from your terminal using[Vercel CLI](https://vercel.com/download):

```
$ vercel
```
