### Running locally

You will need to use the environment variables[defined in  .env.example](https://github.com/vercel-labs/gemini-chatbot/tree/main/)to run Next.js AI Chatbot. It's recommended you use[Vercel Environment Variables](https://vercel.com/docs/projects/environment-variables)for this, but a`.env`file is all that is necessary.

> Note: You should not commit your`.env`file or it will expose secrets that will allow others to control access to your various Google Cloud and authentication provider accounts.
>

1. Install Vercel CLI:`npm i -g vercel`
1. Link local instance with Vercel and GitHub accounts (creates`.vercel`directory):`vercel link`
1. Download your environment variables:`vercel env pull`

```
pnpm install
pnpm dev
```

Your app template should now be running on[localhost:3000](http://localhost:3000/).
