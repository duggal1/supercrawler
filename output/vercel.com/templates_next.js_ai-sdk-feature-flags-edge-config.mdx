To run the example locally you need to:

1. Sign up for accounts with the AI providers you want to use (e.g., OpenAI, Anthropic).
1. Get / Add API keys for each provider.
1. [Set up Edge Config](https://vercel.com/docs/storage/edge-config/get-started)in your Vercel dashboard.
1. Set the required environment variables as shown in the`.env.example`file, but in a new file called`.env`.
1. `npm install`to install the required dependencies.
1. `npm run dev`to launch the development server.
