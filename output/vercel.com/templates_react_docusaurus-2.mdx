### Deploy Your Own

Deploy your own Docusaurus project with Vercel.

[](https://vercel.com/new/clone?repository-url=https://github.com/vercel/vercel/tree/main/docusaurus-2&template=docusaurus-2)

*Live Example:[https://docusaurus-2-template.vercel.app](https://docusaurus-2-template.vercel.app)*

#### How We Created This Example

To get started with Docusaurus on Vercel, you can use the[Docusaurus CLI](https://v2.docusaurus.io/docs/installation#scaffold-project-website)to initialize the project:

```
npx create
-
docusaurus@latest my
-
website classic
```
